Bagaimana cara menggunakan Aplikasi perpustakaan kanesa 
berbasis website pada device yang lain?

1. Pastikan anda sudah memiliki xampp control panel dan vscode
2. Pastikan versi php anda minimal 8.2
3. Buka xampp control panel kemudian nyalakan Apache dan MySQL nya
4. Buka aplikasi perpustakaan kanesa di vscode
5. Buka terminal baru
6. Ketik "php artisan migrate" pada terminal anda, lalu enter
7. Akan muncul pertanyaan "Would you like to create it? (yes/no) [no]"
   Silahkan anda jawab "yes", lalu enter
8. Ketik "php artisan db:seed" lalu enter
9. Pastikan semua proses diatas sudah berhasil.
10. Ketik "php artisan serve", lalu enter
11. tekan ctrl dan arahkan kursor ke url yang ada di terminal, kemudian klik.
   Url => http://127.0.0.1:8000

	Password: 1111
13. Aplikasi sudah bisa anda gunakan dengan maksimal
